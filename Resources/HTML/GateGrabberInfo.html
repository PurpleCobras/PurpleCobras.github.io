Gate Grabber Information

This puzzle game is based on logic gate functionality.

Use the left and right arrow keys or your mouse to move the logic gate input across the bottom.

To change the value of the user input, press the left mouse button:



Collect the gates that satisfy the boolean logic output as they fall.

For more information on logic gates, check out: Logic Gate Lesson »



The visualization on the right side of the screen contains a few key elements:

The logic gates currently falling are shown at the top.

The corresponding speed the gate is falling is shown below.








  • On the dice, the number corresponds to the speed at which the gate will fall

  • The length of the bar also corresponds to the speed of the gate

Hint: There is a possibility that some gates do not have a correct answer. Collect as many correct gates as you can for the highest score!
